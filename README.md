# ✈️ Acerca del proyecto

**SkyReserve Airline System** es una plataforma desarrollada para automatizar y optimizar la gestión de reservaciones aéreas, pasajeros, vuelos y operaciones aeroportuarias mediante herramientas modernas y arquitectura empresarial escalable.

El sistema fue desarrollado para:

- ✈️ Gestionar vuelos comerciales
- 🎫 Administrar reservaciones
- 👥 Gestionar pasajeros
- 💳 Procesar pagos y boletos
- 🛫 Controlar horarios y rutas
- 📊 Generar reportes operativos
- 🌐 Ofrecer acceso multiplataforma

La plataforma permite a aerolíneas y agencias administrar operaciones críticas de manera segura, rápida y eficiente.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/skyreserve-airline-system.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd skyreserve-airline-system
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Configurar entorno

```bash
cp .env.example .env
```

---

## 5️⃣ Configurar base de datos

```env
DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
JWT_SECRET=
```

---

## 6️⃣ Ejecutar migraciones

```bash
npm run migrate
```

---

## 7️⃣ Ejecutar servidor

```bash
npm run dev
```

---

## 8️⃣ Abrir aplicación

```bash
http://localhost:3000
```
